# Cr Encoder

This is a Google Chome extension which allows you to easily encode/decode to/from base64, hexadecimal, URL encoding...

## Available features

* Base64
* Hexadecimal
* URL encoding
* HTML encoding
* Context menu option, for each available conversion, which copies your selection encoded
